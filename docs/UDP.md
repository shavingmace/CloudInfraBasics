# UDP

[[L4-트랜스포트]] [[프로토콜]]이다.

단순하고 빠른 데이터 전송이 목표.
연결을 수립하지 않는다. 
패킷이 도착했는지에 관심 없다. 
[[플로우 컨트롤]] 불가.
짧은 데이터 전송에 적합하다. 
데이터가 깨져도 상관 없는 경우 UDP 전송이 적합하다 - 동영상 스트리밍 등 상황에서.
[[DNS]], [[DHCP]], [[SNMP]], [[TFTP]], ... 등에서 대표적으로 사용된다.
