# 온프레미스 환경


on-premiss environment.

기업이 네트워크나 장소의 소유권을 가지고 있거나 임대하고 자사 소유의 서버를 구축한 환경을 온프레미스 환경이라고 부른다. (서버를 임대한 경우는 온프레미스가 아닌 모양.)

온프레미스의 반대말은 클라우드가 아니다. 반대말은 오프프레미스(off-premiss) 혹은 임대나 공용이다.
