# AS(Autonomous System)

AS. 자율식별번호로 식별되며 동일한 [[라우팅]] 정책으로 관리/운영되는 [[네트워크]]를 말한다. 보통 같은 조직이나 회사에서 관리한다. 

AS는 [[공인 IP 주소]]를 가진 단위가 된다. 

각각의 AS를 구별하는 식별번호가 ASN(자율식별번호)이다. 

[[KISA]]에서 관리하므로 KISA에 신청할 수 있다.
