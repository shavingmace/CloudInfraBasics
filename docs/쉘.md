# 쉘

[[리눅스]]를 포함한 [[링 구조]] OS에 특징적인 부분. 

사용자가 명령어를 실행 시키는 쉘 영역을 만들어낸다. 

쉘은 사용자가 입력한 명령어를 해석하는 [[커널]]과 사용자 사이의 매개 지대이다.


## 리눅스 쉘

Bourne Shell은 매우 초창기의 쉘이다. 

최근엔 주로 bash를 많이 사용한다.
