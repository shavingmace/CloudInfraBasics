# TTY(teletypewriter) 

리눅스 디바이스 드라이브중에서 [[콘솔]]이나 [[터미널]]을 의미한다. 

한 사용자가 여러 터미널을 사용할 수 있다. 

[[표준 입출력]]의 [[리디렉션]]으로 특정 tty에 출력을 내보낼 수도 있다. 

tty 명령어로 지금 사용중인 터미널 번호를 확인 가능하다.
