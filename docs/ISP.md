# ISP

[[인터넷]] 서비스 공급자.

KT, SKT, LG U+ 등이 있다. 

일반적으로 ISP 소속의 [[통신 기사]], [[전기 기사]]가 [[L1-물리]] 계층을 다룬다. 

ISP와 연결되는 회선을 [[인터넷 회선]]이라고 한다.

[[공인 네트워크]]에 연결되는 [[인터넷 회선]]을 제공한다.
