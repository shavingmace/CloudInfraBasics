# 서브넷

커다란 [[네트워크]]를 작게 나눈 네트워크를 말한다.

네트워크를 분할해 직접 통신하는 범위를 좁히고 [[방화벽]]을 설정해 [[보안]]을 강화하는 것을 목적으로 한다. 네트워크의 서브넷을 나누면 서브넷에 따라 특정 서브넷을 공개하거나 비공개 할 수 있다.

서브넷끼리 통신하려면 통상적으로 [[라우팅]]이 필요하다.
