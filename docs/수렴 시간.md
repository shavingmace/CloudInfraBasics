# 수렴 시간

convergence time

네트워크 장비에서 [[동적 라우팅 프로토콜]]이 작동할 때 네트워크에 변화가 생기면 [[라우팅]] 설정을 변경하는데에 걸리는 시간.

수렴 시간이 느리면 [[라우팅 루프]]가 발생할 확률이 높다.
