# NAT

Network Address Translation. 

네트워크 주소 변환 기술이다. 흔히 [[공유기]]나 방화벽은 NAT로 [[사설 IP 주소]]를 사용하게 해준다. 로컬 네트워크 외부로 나갈 때에는 NAT 장비가 [[공인 IP 주소]]로 변환해 주어 인터넷에 접속할 수 있다.
