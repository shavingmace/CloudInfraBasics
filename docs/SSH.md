# SSH (Secure Shell)

네트워크 상의 다른 디바이스에 로그인 or 원격 시스템에서 명령을 실행하는 등을 수행하는 응용 프로그램 혹은 [[프로토콜]].
데이터전송, [[원격제어]]에 많이 사용된다.
주로 22번 포트를 사용한다.

한쌍의 key([[공개키]]/[[개인키]]) 를 통해 원격지 디바이스와 [[인증]]을 거친다.
