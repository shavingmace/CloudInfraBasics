# 프로토콜


소통 규약.

웹 기술의 관점에서 프로토콜의 목록은

[[FTP]] 등이 있다.

현대 네트워크에서는 몇몇 층에서 프로토콜의 논쟁이 냉각되고 프로토콜의 표준이 거의 정해졌다. 그런 이유로 직관적 이해에서는 [[OSI 7계층]] 모델보다 [[TCP-IP]] 모델이 더 유용하다.
