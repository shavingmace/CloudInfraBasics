# 마크업 언어

markup language.

본문과 구분할 수 있는 마크mark를 추가할 수 있는(annotation) 시스템.

인쇄술 시대에도 작가와 편집자는 조판typesetting을 담당하는 인쇄공에게 자신이 원하는 사항을 묘사해 전달하기 위해 **조판부호**를 사용했다. 

**마크업 언어는 최신의 조판부호다.**


대표적 마크업 언어로는 
- GML
- SGML
- [[XML]]
- [[HTML]]
등이 있다. 아래 두 언어는 SGML을 뿌리로 둔다. 

참조: [[마크다운]]
